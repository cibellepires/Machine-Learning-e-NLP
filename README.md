# Machine-Learning-e-NLP
🤖 Estudos de Machine Learning e NLP

Repositório dedicado ao estudo e experimentação de Machine Learning (ML) e Processamento de Linguagem Natural (NLP) utilizando Python e bibliotecas do ecossistema de ciência de dados.

O objetivo deste repositório é organizar experimentos, implementações e análises de modelos aplicados a dados estruturados e textos, explorando desde conceitos fundamentais até técnicas modernas.

🎯 Objetivos

Estudar conceitos fundamentais de Machine Learning

Implementar algoritmos clássicos e modelos modernos

Explorar técnicas de Processamento de Linguagem Natural (NLP)

Experimentar diferentes representações de texto e embeddings

Praticar análise de resultados e avaliação de modelos

🛠️ Tecnologias Utilizadas

Python

NumPy

Pandas

Scikit-learn

PyTorch / Transformers

Matplotlib / Seaborn
